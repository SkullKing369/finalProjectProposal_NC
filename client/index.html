cowDraw V. 2.0

Team members: Nathan Crouch(author)
App Overview: They will enter the name of a text file and reads it out, then draws
a regular sized cute cow and then a smaller cute cow
Technology Overview: Uses the Python language and the turtle library
Proposal Last Updated: 04/20/2023
Feature List: Accepts user input, reads from file, prints text from file, draws two cows.
Future Feature List: Perhaps allow user input to decide what kind of cow to draw.
Basic Technical Features: Accepts user input, draws onto a screen
Advanced Technical Features: Draws advanced/irregular shapes, reads from given file
